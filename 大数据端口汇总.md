## Flink

- 8081 ： WEB UI 端口(和spark同时使用会有冲突)
- **Spark集群后再启动Flink会发现8081端口被Spark占用了**
- **Kafka的zookeeper和Flink有冲突。**

```shell
vim flink-conf.yaml 
修改或添加（注意:后有一个空格，否则无法生效）：
rest.port: 8082
```
